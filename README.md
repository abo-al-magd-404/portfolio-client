# Portfolio Client

## Tech Stack
- **Frontend:** React, HTML, CSS
- **State Management:** Redux
- **Routing:** React Router
- **Testing:** Jest, React Testing Library
- **Deployment:** Netlify

## Project Structure
```
portfolio-client/
├── public/
├── src/
│   ├── components/
│   ├── pages/
│   ├── redux/
│   └── App.js
├── .env
└── package.json
```

## Pages
- **Home:** The landing page for the portfolio.
- **About:** Information about the project and the author.
- **Projects:** Showcase of work done.
- **Contact:** A form to reach out to the author.

## Admin Access
To access admin features, visit `/admin` with valid credentials. You can create an admin user in the database.

## Environment Variables
- `REACT_APP_API_URL`: Base URL for API calls
- `REACT_APP_CONTACT_EMAIL`: Contact email for the portfolio

## Running Instructions
1. Clone the repository: `git clone https://github.com/abo-al-magd-404/portfolio-client.git`
2. Navigate to the project directory: `cd portfolio-client`
3. Install dependencies: `npm install`
4. Start the development server: `npm start`

## Deployment Notes
This project is designed to be deployed on Netlify. Please ensure that the environment variables are correctly set in the Netlify dashboard.
